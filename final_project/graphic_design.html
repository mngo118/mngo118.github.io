Expressive Word Project

For this project we were given a list of words and had to create an image to represent the word, using the word. Meaning that we weren't allowed to draw or create other objects. We were only allowed to use parts of the typed out word. I chose strain and tolerate.

Logo Project

For this project we were tasked to create a logo for a new company. I was given a new boba store in Bellevue to create a logo and name for.

Business Card Project

For this project we were instructed to create a business card for our personal use/brand. This card has lots of geometric minimalism influence. I wanted to make this card very approachable and fun, so I made the card playful with bright pastel colors.
