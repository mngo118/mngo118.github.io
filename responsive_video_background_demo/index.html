Responsive CSS Video Background

This tutorial will teach you how to add a video background to your website that will respond to different screen sizes.

You can check out the finished product here. But let's get started

First you'll want to build your base website. I just made a centered div with text with this code:

        
            IN HTML DOCUMENT: 
            <div id="textdiv">
                <h1>Lorem ipsum</h1>
                <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
                Curabitur porta dictum turpis, eu mollis justo gravida ac. 
                Proin non eros blandit, rutrum est a, cursus quam.</p>
            </div>

            IN CSS DOCUMENT:
                html,
                body {
                    width: 100vw;
                    height: 100vh;
                    margin: 0;
                    font-family: Arial, Helvetica, sans-serif;
                }
                #demo-text {
                    text-align: center;
                    margin: 5% 20%;
                }

I converted the youtube video bellow to a mp4 (you can use any online converter, I used 320YTMP3).

I first added the video to the initial html file. I added autoplay to make the video start playing right away, muted to play without sound, and loop to play for an infinite amount of times

        
            <video id="videoBG" poster="poster.JPG" autoplay muted loop>
                <source src="footage1.mp4" type="video/mp4">
            </video>

In the css file I set the position to fixed to keep the video there, and set z minus 1 to push the video to the background

        
            #videoBG {
                position:fixed;
                z-index: -1;
            }

It seems to work! But it isn't responsive (doesn't change when you resize the screen). To make it responsive I added this to the CSS document:

        
            #videoBG {
                position:fixed;
                z-index: -1;
            /*not work if the screen ratio is below 16/9*/
                width:100%;     
                height: auto;
            }

We need to keep the width to 100% and scale the height. We can achieve this by using min-aspect-ratio 16/9.

        
            @media (min-aspect-ratio: 16/9) {
                #videoBG {
                    width:100%;
                    height: auto;
                }
            }

Then we'll keep the height 100% when max-aspect ratio is bellow 16/9

        
            @media (max-aspect-ratio: 16/9) {
                #videoBG { 
                    width:auto;
                    height: 100%;
                }
            }
